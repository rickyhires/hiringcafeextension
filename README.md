# HiringCafe+

A personal Chrome extension (Manifest V3, no build step) that supercharges [hiring.cafe](https://hiring.cafe) job search: cross-check badges, company ratings & intel, ATS info, job signals, smart filters/sort, tracking with real account write-back, alerts, and a keyboard/command-palette layer — all built on the site's own rich job data. **v1.1.0.**

## Install / update
1. Open `chrome://extensions`
2. Enable **Developer mode** (top right)
3. **Load unpacked** → select this folder (or click the **↻ reload** icon on the HiringCafe+ card after updating)
